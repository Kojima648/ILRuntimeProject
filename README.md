此工程的主要流程：

1. 设置热更配置表数据结构
2. 生成热更包几热更配置表放到服务器
3. 下载热更配置表读取需要热更文件
4. 下载热更文件
5. 游戏中加载热更

其他：

	1. 自动生成热更包及自动生成热更配置表
 	2. 本地记录下载热更版本
 	3. 热更下载中断等处理
 	4. 热更资源下载后校验
 	5. 热更回退
 	6. 资源加密，解密
 	7. 资源解压
